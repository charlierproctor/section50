|-------------------|---------------|-------------------|

```c 
// takes a sorted array `values` of length `n` and return a boolean
// indicating whether or not `value` is in `values`.
bool search(int value, int values[], int n)

search(5, {1,2,3,4,5}, 5) => true
search(2, {1,3,5,7}, 4) => false
```
